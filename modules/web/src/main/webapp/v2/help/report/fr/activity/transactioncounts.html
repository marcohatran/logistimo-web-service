La transaction compte

Le rapport sur le nombre de transactions fournit un aperçu de l'activité de transaction entre les entités. La métrique de base est un décompte du nombre de transactions sur tous les types ou sur un type de transaction particulier, tel que l'émission, la réception, le décompte des stocks, la suppression ou le transfert.

Les métriques de ce rapport, décrites ci-dessous:

  • Nombre de transactions: Cette métrique représente le nombre de transactions - toutes ou d'un type particulier - effectuées au cours d'une période donnée.
  • Nombre moyen de transactions par entité: Cette métrique est calculée en divisant le nombre total de transactions par le nombre d'entités pour obtenir un nombre de transactions moyen par entité.
  • % d'entités avec des transactions: Cette métrique représente la proportion d'entités qui ont au moins une transaction dans la période donnée. Ceci est calculé en divisant le nombre d'entités avec au moins une transaction dans la période donnée par le nombre total d'entités dans cette période.
